# Notes

- This is a convenient version from [CanItEdit](https://github.com/nuprl/CanItEdit) and [MultiPL-E](https://github.com/nuprl/MultiPL-E).

- Install environment `canitedit`:

```
conda create -n canitedit python==3.10.12
pip install -r requirements.txt
```

- You can directly use this directory for local evaluation, where we only update `pass_k.py` and `requirements.txt`.
